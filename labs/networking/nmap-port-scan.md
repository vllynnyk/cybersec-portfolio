# Port Scanning and Web Service Discovery

## Objective
Identify open ports and discover a web service.

## Steps
- Performed port scan using nmap
- Found 6 open ports
- Identified HTTP service on port 8008
- Accessed the web server via browser

## Commands
- nmap -sV MACHINE_IP

## Result
- 6 open TCP ports detected
- Web server found on port 8008
- Retrieved flag from web page

## Key Takeaways
- How to perform port scanning
- How to identify web services on non-standard ports
